TORNEO-APP (Liga / Copa / Copa con grupos)

Cómo usar
1) Abrí index.html con doble click (Chrome/Edge/Opera).
2) Menú:
   - Crear torneo: agregás participantes, elegís ligas, sorteás equipos (sin repetir), generás fixture y guardás.
   - Torneos / Historial: cargás resultados partido a partido, mirás tabla o llave, y cuando termina se guarda el campeón.
   - Equipos y ligas: podés corregir/agregar/quitar equipos por ascensos/descensos.

Backup
- "Exportar backup" te descarga un JSON con torneos + ranking.
- "Importar backup" restaura ese JSON en el navegador donde lo importes.

Notas
- Todo se guarda en tu navegador (localStorage). Si abrís la app en otra PC o navegador, no se comparte.
- Si tenés muchos participantes (ej 30) y elegiste pocas ligas, asegurate de que haya suficientes equipos para asignar.
